## About This Repository

**C# Practice Projects** is a console-based project collection created to improve C# and .NET development skills through practical mini-projects.

This repository focuses on writing clean, organized, and beginner-friendly C# code while learning real programming concepts.

---

## Features

| Module | Description |
|---|---|
| Smart Calculator | Perform basic and advanced calculations |
| To-Do Manager | Add, update, complete, and delete tasks |
| Expense Tracker | Track income, expenses, and balance |
| Contact Book | Save and search contact details |
| Notes Manager | Create and manage personal notes |
| Password Generator | Generate strong random passwords |
| Number Guessing Game | Play a simple console-based game |
| Text Analyzer | Count words, characters, lines, and more |
| Quiz Game | Practice C# concepts through questions |
| Number Toolkit | Work with prime numbers, factorials, and other number operations |

## Tech Stack

- **Language:** C#

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/jeetyadav9/csharp-practice-projects.git
```

### Open the Project Folder

```bash
cd csharp-practice-projects
```

### Run the Application

```bash
dotnet run --project src/CSharpPracticeProjects/CSharpPracticeProjects.csproj
```

### Run Tests

```bash
dotnet test
```
